vetiver.pin_read_write.vetiver_pin_read

vetiver.pin_read_write.vetiver_pin_read#

vetiver.pin_read_write.vetiver_pin_read(board, name: str, version: Optional[str] = None) → vetiver.vetiver_model.VetiverModel#

Read pin and populate VetiverModel

Parameters

  • board – A pin board, created by pins.board_folder() or another board_ function.

  • name (string) – Pin name

  • version (str) – Retrieve a specific version of a pin.

Return type

vetiver.VetiverModel

Notes

If reading a board from RSConnect, the board argument must be in “username/modelname” format.
